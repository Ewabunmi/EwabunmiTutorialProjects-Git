WHAT IS A FRONT-END DEVELOPER?

Front-end web development, also known as client-side development is the practice of producing HTML,CSS and JavaScript for a website or web application so that a user can see and interact with them directly.The challenges asociated with front end development is that the tools and techniques used to create the front end of a website change constantly and so the developer needs to constantly be aware of how the fields is developing.

The objective of designing a site is to ensure that when the users open up the site they see the information in a format that is easy to read and relevant.They need to ensure that the site comes up correctly in different systems and different devices, which requires careful planning on the side of the developer.
